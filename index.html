The Tool

PixelWrench is used to design worlds and character sets. It's made for visualising your builds and outfits before spending anything in-game, similar to PWTools by ColdUnwanted. Built for power users who want absolute control over their designs.

01

World Planner

Design Worlds and test out gear in a free environment to see what looks best. High-density interface for professional builders.

02

Game Commands

WIP API that uses in-game chat commands to download world maps. Seamlessly import your in-game creations for rapid iteration.

03

Schematics

Blueprint system like Litematica. Export your PixelWrench work and use it as an interactive guide while building inside PixelWorlds. Never miss a block again.
